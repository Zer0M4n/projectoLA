🐍 Manual del Lenguaje LA

Manual de Referencia del Lenguaje LA

Este documento describe la sintaxis, las palabras clave y el funcionamiento del intérprete LA, un lenguaje sencillo diseñado para la ejecución de scripts con control de flujo y funciones.

1. Variables y Tipos de Datos

Las variables deben ser declaradas usando la palabra clave `var` y son de tipado dinámico.

1.1 Declaración y Asignación

Característica Sintaxis Ejemplo
Declaración var <nombre> = <valor> var edad = 20
Números (int/float) Se admiten valores enteros y decimales. var precio = 15.99
Cadenas (Strings) Deben ir entre comillas dobles. var nombre = "César"

1.2 Operaciones y Funciones Integradas

Se soportan operaciones aritméticas básicas y concatenación:

var x = 10
var y = 5
print(x + y)    
print("Resultado: " + str(x * y))

Para la conversión de tipos en la salida, use las funciones integradas seguras: str(), int(), y float().

---

2. Control de Flujo: Condicionales

El control de flujo se maneja con la estructura if/else. La **indentación** (sangría) es obligatoria y define los bloques de código.

2.1 Estructura IF/ELSE

var edad = 22
if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")

Se admiten los operadores de comparación habituales: ==, !=, >, <, >=, <=.

---

3. Funciones

Las funciones se definen con la palabra clave func y pueden incluir parámetros, así como retornar valores.

3.1 Definición y Retorno

Use la instrucción return para devolver un valor. Esto finaliza la ejecución de la función inmediatamente.

func sumar(a, b):
    return a + b

var resultado = sumar(10, 5)
print("La suma es " + str(resultado))

3.2 Scoping (Alcance de Variables)

El lenguaje soporta funciones anidadas y jerarquía de variables:

  • **Parámetros y Variables Locales:** Tienen prioridad y solo existen dentro de la función.
  • **Variables Globales:** Todas las variables declaradas fuera de las funciones son accesibles por cualquier función.
func cuadrado(n):
    return n * n

func calcular_area(lado1, lado2):
    var sq1 = cuadrado(lado1) 
    var sq2 = cuadrado(lado2)
    return sq1 + sq2

print(calcular_area(3, 4))
---

4. Impresión de Salida

Utilice la función print() para mostrar el resultado de cualquier expresión evaluada en la consola.

var saludo = "¡Hola!"
print(saludo)
print(15 * 2)
